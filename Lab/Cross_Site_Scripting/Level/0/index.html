Perfecting your payload

ย้อนกลับ
LEVEL ONE

โดยในตัวแลปของเราจะใช้ JavaScript เป็นคำสั่งเพื่อใช้ในการแสดงฟังชั่นการแจ้งเตือน ด้วย String HKA, เดี๋ยวไปดูตัวอย่างกันครับ

<script>alert('HKA');</script>
